# Projet de Publication sur Discord

Ce projet est une application web permettant aux utilisateurs d'envoyer des messages sur Discord en utilisant des webhooks.

## Fonctionnalités

- Formulaire de publication sur Discord.
- Envoi de messages à Discord en utilisant des webhooks.
- Affichage de notifications pour indiquer si le message a été envoyé avec succès ou s'il y a eu une erreur.
- Tutoriel sur la création d'identifiants de webhook.
